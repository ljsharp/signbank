{% blocktrans %}Who can create an account?{% endblocktrans %}

{% blocktrans %}Only create an account if you have been in contact with the staff beforehand.{% endblocktrans %}

{% blocktrans %}Even though anyone can create an account, one needs to get additional user permissions to be able to do anything. These permissions can only be set by the staff, and therefore you should contact the staff before creating an account.{% endblocktrans %} {% blocktrans %}Permissions are typically granted to persons doing sign language research or corpus annotation.{% endblocktrans %}

{% blocktrans %}What can you do with an account?{% endblocktrans %}

{% blocktrans %}Depending on the level of user permissions, you can see more detailed data about glosses, add data, and edit data.{% endblocktrans %} {% blocktrans %}Please note that you can use the lexicons even without creating an account.{% endblocktrans %} {% blocktrans %}It is possible to use the lexicons for example in teaching and studyig of sign languages, and you can also download the videos according to the Creative Commons BY-NC-SA license.{% endblocktrans %}

  • {% blocktrans %}You can apply for permissions to view more detailed information about specific lexicons on{% endblocktrans %} {% blocktrans %}the Lexicons page{% endblocktrans %}, {% blocktrans %}which you can access once you have created an account (from the menu: Manage->Lexicons){% endblocktrans %}.
  • {% blocktrans %}Contact the site maintainer or the site staff.{% endblocktrans %}
